Map

Loop over an array of items and transform the results. This is an alias to the select function.
Here you can see that we loop over the range of numbers and pow(2) each item into a new array result. The variable 'source' now contains the answers. 

var source = Rx.Observable.range(1, 3)
    .map(function (x, idx, obs) {
        return x * x;
    });

source.forEach(function(x) {
   console.log(x);
});
